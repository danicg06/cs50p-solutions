# Week 2 — Loops

This folder contains a collection of programming exercises from Week 2 of CS50's Introduction to Programming with Python. 

This week introduced fundamental concepts of loops along with techniques for user input validation. The exercises required the use of `while` and `for` loops, as well as the `dict` data structure and the built-in `len()` function to solve the problems.

The detailed problem statements and implementation notes can be found inside the comments at the top of each `.py` file.

---

### Requirements
* Python 3.x installed on your system.
