# Week 3 — Exceptions ❗
This folder contains a collection of programming exercises from Week 3 of CS50's Introduction to Programming with Python. 
This week introduced advanced exception handling and error behaviour control, especially in response to user's input. This was simplified
by the use of `try` and `except`, combined with `pass` or `break`.
The detailed problem statements and implementation notes can be found inside the comments at the top of each `.py` file.
---
## 🛠️ Requirements and Execution
### Requirements
* Python 3.x installed on your system.
### Execution Command
To run any of the scripts, navigate to this directory and execute:
```bash
python script_name.py
```
## 📂 Files
* `fuel.py` — conversion from a fraction to a percentage.
* `grocery.py` — multiple item input to list with the quantity of each item.
* `outdated.py` — M/D/YYYY or Month Day, Year date to YYYY-MM-DD date.
* `taqueria.py` — buying items from a taqueria.
