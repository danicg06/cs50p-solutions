# Week 6 — File I/O 📁

This folder contains a collection of programming exercises from Week 6 of CS50's Introduction to Programming with Python.

This week focused on **file input and output**, teaching how to read from and write to external files such as text files and CSVs. It covered managing resources safely with the `with` statement, cleaning and splitting data lines, sorting information, and handling multiple file formats like images.

The detailed problem statements and implementation notes can be found inside the comments at the top of each `.py` file.

---

## 🛠️ Requirements and Execution

### Requirements
* Python 3.x installed on your system.
* Required external libraries (such as `Pillow` for image processing) installed via `pip`.

### Execution Command
To run any of the scripts, navigate to this directory and execute:
```bash
python script_name.py
