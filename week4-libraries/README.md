# Week 4 — Libraries 📚

This folder contains a collection of programming exercises from Week 4 of CS50's Introduction to Programming with Python. 

This week focused on the power of libraries to extend Python's functionality. The exercises covered how to import modules (both built-in and external) to handle tasks like text-to-emoji conversion, formatting output with specific fonts, and managing lists of names, all while becoming familiar with the pip package manager.

The detailed problem statements and implementation notes can be found inside the comments at the top of each `.py` file.

---

## 🛠️ Requirements and Execution

### Requirements
* Python 3.x installed on your system.
* Necessary libraries installed via `pip` (e.g., emoji, pyfiglet).
  
### Execution Command
To run any of the scripts, navigate to this directory and execute:
```bash
python script_name.py
```

## 📂 Files
* `adieu.py` — prompts the user for names and bids them adieu with proper formatting.
* `emojize.py` — converts text aliases into emoji characters.
* `figlet.py` — prints text using different fonts provided by the pyfiglet library.
