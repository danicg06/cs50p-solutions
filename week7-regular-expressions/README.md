# Week 7 — Regular Expressions 🔍

This folder contains a collection of programming exercises from Week 7 of CS50's Introduction to Programming with Python.

This week focused on **regular expressions** using Python's built-in `re` library. It explored how to define patterns to validate, search, and extract specific text formats such as email addresses, URLs, log files, and formatted identifiers, using features like capture groups and flags.

The detailed problem statements and implementation notes can be found inside the comments at the top of each `.py` file.

---

## 🛠️ Requirements and Execution

### Requirements
* Python 3.x installed on your system.

### Execution Command
To run any of the scripts, navigate to this directory and execute:
```bash
python script_name.py
```

## 📂 Files
* `numb3rs.py` — validates IPv4 addresses to ensure they follow correct numerical ranges.
* `watch.py` — extracts shortened YouTube URLs from HTML embed strings and converts them into cleaner shareable links.
* `um.py` — counts occurrences of the word "um" as a standalone word in text case-insensitively.
* `response.py` — validates email addresses using a robust external validator library.
* `working.py` — parses and converts 12-hour time formats into 24-hour time strings.
