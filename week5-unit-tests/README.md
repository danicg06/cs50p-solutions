# Week 5 — Unit Tests 🧾

This folder contains a collection of programming exercises from Week 5 of CS50's Introduction to Programming with Python. 

This week focused on the importance of testing code. It introduced pytest as a tool for verifying that functions behave as expected by running automated tests. The exercises emphasized how to structure test files, use assertions to validate output, and catch expected exceptions, ensuring code remains robust and reliable.

The detailed problem statements and implementation notes can be found inside the comments at the top of each `.py` file.

---

## 🛠️ Requirements and Execution

### Requirements
* Python 3.x installed on your system.
* `pytest` library installed.
  
### Execution Command
To run any of the scripts, navigate to this directory and execute:
```bash
pytest test_file.py
```
## 📂 Files
* `test_bank.py` — unit tests for a function that determines the greeting bonus.
* `test_fuel.py` — unit tests for fuel gauge gauge conversion and error handling.
* `test_plates.py` — unit tests for vanity plate validation logic.
* `test_twttr.py` — unit tests for a function that removes vowels from text.
